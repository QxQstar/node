# 入口文件app.js
根目录下的app.js是入口文件，在这个文件中完成连接数据库，创建服务器，配置模板引擎，设置基本中间件的操作。
# public文件夹
所有的静态文件都位于public文件夹下。今天文件有js文件，css文件及img文件
# config文件夹
config目录下只有一个route.js文件，所有的路由都在这个文件中。
# app文件夹
在app文件夹中有四个文件夹，分别是views，controls，models，schemas。
在views文件夹中是模板文件，control文件夹中是路由的中间件，models文件夹中是mongoose的Model，
schemas文件夹中是mongoose的Schema。
